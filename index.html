🔧 Blattner Energy REQ Tracker — First-Time Setup

This tracker stores data in Firebase so everyone on your team shares the same data in real time. Follow the 3 steps below — it takes about 5 minutes.

1Create a free Firebase project

Go to console.firebase.google.com → Click Add project → Name it ejde-tracker → Disable Google Analytics → Click Create project.

Then: In the left menu → Build → Realtime Database → Create database → Choose United States → Start in Test mode → Enable.

2Get your Firebase config

In Firebase console → Click the ⚙️ gear icon → Project settings → Scroll down to "Your apps" → Click </> Web → Register app (name: tracker) → Copy the firebaseConfig object.

Paste it here (the whole block from const firebaseConfig = { to the closing };):

3Add your team members

Enter the names of everyone who will use this tracker (so "Prepared By" and other dropdowns are correct).

4Share the URL with your team

After saving, host this HTML file anywhere and share the link:

• Easiest: Go to netlify.com/drop → drag this HTML file → get a URL in 30 seconds.

• SharePoint: Upload to a SharePoint page as a web part.

Everyone opens that URL — all data is shared and syncs in real time. ✅
